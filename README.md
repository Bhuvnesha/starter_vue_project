# Laravel 12 + Vue 3 + Inertia Starter

## 🚀 Tech Stack
- Laravel 12
- Vue 3
- Inertia.js
- Tailwind CSS

## ⚙️ Installation


## Installation
git clone https://github.com/your-username/laravel-vue-inertia-starter.git
cd laravel-vue-inertia-starter
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev


## Features
